Architecture & Privacy

This system utilizes a Zero-Trust Workflow. Unlike traditional converters, your files are never transmitted to a remote server. Instead, the system uses:

  • Canvas API Rendering: For web formats, the browser repaints the pixel matrix directly into the new format buffer.
  • HEIC/RAW Bridge: We use specialized logic to convert proprietary Apple and DSLR formats into renderable web assets locally.

Frequently Asked Questions

Is there a file size limit?

The limit is determined by your system's available RAM. Most devices handle up to 100MB flawlessly.

Are my photos stored?

No. Processing is 100% ephemeral. Once you close this tab, all data is purged from memory.
