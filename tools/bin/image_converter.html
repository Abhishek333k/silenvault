Architecture & Privacy

This system utilizes a Zero-Knowledge Workflow. Your files are never transmitted to a remote server. Instead, the system uses:

  • Canvas API Rendering: For web formats, the browser repaints the pixel matrix directly into the new format buffer.
  • WASM RAW Engine: For CR2 and DNG files, a high-performance WebAssembly module extracts high-fidelity data locally.

Frequently Asked Questions

Is there a limit on file size?

The limit is determined by your system's available RAM. Most modern devices handle up to 100MB per file flawlessly.

Are my photos stored?

No. Once you close this tab, all processed data is cleared from your browser's temporary memory.
